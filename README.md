# YouTube Channel Tracker & Analytics

A simple web application that allows you to search YouTube channels, track their statistics, and compare their growth in real time.

## Features

- Search YouTube channels
- Track subscriber count
- Monitor view count
- Compare multiple channels
- See live updates of channel statistics
- Simple and lightweight web interface

## How It Works

This project uses the **YouTube Data API v3** to fetch channel data and display statistics in real time.

The application stores short-term history in memory to visualize changes in subscribers and views.

## Requirements

- Python 3.8+
- Flask
- Requests
- YouTube Data API key
